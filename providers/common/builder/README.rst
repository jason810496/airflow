Package ``apache-airflow-providers-common-builder``

Release: ``1.0.0``

``Common Provider Builder``

Provider package
================

This is a provider package for ``common.builder`` provider.
All classes for this provider package are in ``airflow.providers.common.builder`` python package.

Installation
============

You can install this package on top of an existing Airflow installation (see
``Requirements`` below) via ``pip install apache-airflow-providers-common-builder``.

Requirements
============

==================  ==================
PIP package         Version required
==================  ==================
``apache-airflow``  ``>=3.2.0``
``textual``         ``>=6.2.0``
==================  ==================
